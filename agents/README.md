# Agent Catalog

This top-level directory documents the AI agent landscape for repository consumers.

Runtime source code lives in `backend/app/agents/`.

Current production agents:

- critical escalation
- campaign detection
- duplicate detection
- response drafting with RAG
- repeat requester analysis
- medicine supply routing
- care quality routing
- sanitary and epidemiological routing

Repository roadmap items for this area include per-agent runtime controls, testing, quality metrics, and retraining workflows.
