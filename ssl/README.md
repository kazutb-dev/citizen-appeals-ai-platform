# ssl/

Каталог для TLS-сертификатов исследовательского стенда.

**Сертификаты и приватные ключи НИКОГДА не коммитятся в репозиторий.**
Прежний `private.key`/`fullchain.pem` были удалены как утёкшие — перевыпустите сертификат.

При деплое смонтируйте сертификаты сюда (volume/bind) или передайте через секреты:

- `fullchain.pem` — цепочка сертификатов
- `private.key` — приватный ключ

nginx (`nginx/nginx.prod.conf`) читает их из `/ssl/`.
