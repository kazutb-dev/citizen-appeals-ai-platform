# Changelog

All notable changes to this project are documented in this file.

The format is based on Keep a Changelog and the project uses semantic versioning for tagged releases.

## [Unreleased]

### Added

- Open source repository governance files, contribution policy, and security policy.
- Local-only LLM runtime support in Docker Compose with `ollama` and model bootstrap.
- Unified intake schema for source channel, geolocation, and intake idempotency.
- Mandatory location selection for citizen appeal submission.

### Changed

- Removed cloud Anthropic provider paths from the backend.
- Expanded repository standards for formatting, linting, and hook automation.

### Planned

- Replace mock integration center providers with a real integration layer.
- Consolidate dashboards and simplify navigation for production operations.
- Reduce social monitoring scope to Telegram and Instagram only.
